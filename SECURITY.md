# Security Policy

## Supported Versions

We provide security updates for the latest released version of IIoT Edge Gateway.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by:

1. Creating a [Security Advisory](https://github.com/KuzinHouse/IIoT-Edge-Gateway/security/advisories/new)
2. Or emailing directly to: dimiam.dimiam@gmail.com

We will acknowledge your report within 48 hours and provide a fix within 7 days.

## Security Best Practices

- Keep your gateway behind a firewall
- Use TLS/SSL for all external connections
- Rotate API keys regularly
- Follow the principle of least privilege (use minimal role assignments)
- Enable audit logging for compliance
- Keep dependencies up to date
- Disable unused protocols and ports
- Use strong password policies (minimum 12 characters)
